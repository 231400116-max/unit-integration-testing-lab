## Project Description
This project demonstrates unit testing and integration testing using pytest.
A simple banking application is tested to verify individual functions and
their combined workflows.

## How to Run Tests
Install dependencies:
pip install -r requirements.txt

Run all tests:
pytest

Run with verbose output:
pytest -v

Generate HTML report:
pytest --html=report.html -v

## GitHub Actions CI
GitHub Actions automatically runs unit and integration tests on every push
and pull request to ensure code quality.
